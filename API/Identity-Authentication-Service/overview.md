# JDCloud IAS


## 简介
京东云联合登陆


### 版本
v1


## API
|接口名称|请求方式|功能描述|
|---|---|---|
|**appDetail**|GET|运营后台获取应用详情|
|**apps**|GET|运营后台查询app|
|**createApp**|POST|创建应用|
|**deleteApp**|DELETE|删除应用|
|**getApp**|GET|获取应用|
|**getApps**|GET|获取账户下所有应用|
|**state**|GET|运营后台获取应用状态|
|**updateApp**|PUT|更新应用（只传需要变更的参数，不传的参数不会更新）|
