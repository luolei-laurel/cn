# enableKey


## 描述
启用当前状态为`已禁用`的密钥

## 请求方式
PATCH

## 请求地址
https://kms.jdcloud-api.com/v1/key/{keyId}:enable

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**keyId**|String|True| |密钥ID|

## 请求参数
无


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String|请求ID|


## 返回码
|返回码|描述|
|---|---|
|**200**|OK|
|**404**|Not found|
