## 添加准备

购买web应用防火墙服务后，您需要将您的网站接入web应用防火墙来进行防护。

接入WAF之前，您需要了解以下内容：

1、 对于中国大陆地区的WAF实例，添加的网站需通过工信部备案。

2、 网站源站服务器的IP需填写正确。

3、 使用https提供服务的网站需要准备该域名绑定的证书和私钥信息并上传。

4、 接入web应用防火墙需修改DNS记录，请提前准备DNS解析记录的编辑权限，熟悉对应的域名服务商的控制台操作方法，以便修改解析记录。 

具体操作步骤：

1、 添加网站配置

2、 放行WAF回源IP

3、 本地验证

4、 修改DNS解析
