# 源代码配置

1、京东云代码托管

集成了京东云代码托管（code.jdcloud.com）的代码源，无需二次认证，直接打通用户在代码托管的代码仓库.

仓库地址：远程仓库的URL地址,与 git clone 命令的地址相

代码分支：选择代码分支

2、GitHub社区版集成

GitHub社区版插件可以帮助用户直接绑定GitHub社区版账户，自动列出您的代码仓库及其对应的分支或Tag

操作步骤

  * 点击 绑定GitHub账号

  * 页面会跳转至GitHub的登录页面

  * 登录成功后会跳转到应用页面
  

3、通用Git仓库集成

仓库地址：远程仓库的URL地址,与 git clone 命令的地址相

代码分支：填写代码分支
