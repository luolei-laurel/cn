# 产品规格

分布式网络负载均衡采用全可用区分布式部署架构，无负载均衡实体存在，转发性能无瓶颈，可以应对客户大规模业务流，无需扩展。在分布式网络负载均衡所在子网只需预留一个私网IP，该IP作为VIP。
